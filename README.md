# Sumplete QML Game
 
## Что это?
 Это наивная реализация широко известной игры на математику.
Данный проект написан на C++ c использованием Qt 5.15.13 и QML.

## Текущие возможности

Первая версия содержит:
- простейший цикл игры для поля размером 9х9 и числами [1-9],
-  автоматическое сохранение и загрузка текущей игры,
- проверка на выигрыш в текущей игры.

Первая версия **не** содержит:
- рекламу,
- платные возможности,
- обращения к посторонним сетевым ресурсам,
- имя автора.
